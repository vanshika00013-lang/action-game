# 🎮 Action Game (HTML, CSS, JavaScript)

A simple action shooting game made using pure HTML, CSS, and JavaScript.

## 🚀 Features
- Player movement (W, A, S, D)
- Shooting with mouse click
- Enemy AI (chasing player)
- Boss fight with health bar 💀
- Powerups:
  - ❤️ Health
  - ⚡ Speed
  - 🔥 Rapid Fire
- Score system
- Game Over screen
- Restart with ENTER

## 🎮 Controls
- Move: W A S D
- Shoot: Mouse Click
- Restart: ENTER

## 🧠 Tech Used
- HTML Canvas
- CSS
- JavaScript (Vanilla)
